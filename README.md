Responsive Order Book example with NextJS using WebSocket
## Getting Started

Copy `.env.example` as `.env.local`

Install

```bash
yarn
```

Run development server

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
